# Sand Process Management System

A comprehensive operations management system for sand processing operations in Vaca Muerta, Argentina.

## Features

- Order Management
- Logistics & Traceability with GPS Tracking
- Quality Control
- Billing & Payment
- Inventory Management
- Fleet Management
- Customer Portal
- Bilingual Support (English/Spanish)

## Getting Started

### Installation

```bash
npm install
```

### Run Development Server

```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

## Tech Stack

- React 18 with TypeScript
- Material-UI
- React Router
- i18next for internationalization
- Leaflet for maps
- Recharts for data visualization
